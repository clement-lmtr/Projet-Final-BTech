# README — Projet Solo


## 1. Informations générales

- **Nom / Prénom :** Lemettre Clément

- **Nom du projet / du site :** Sweet Navy Tatoo

- **Thème choisi :**  Site de réservation

- **Public cible :** Tout public à partir de 18 ans / 16 ans avec autorisation parental.

---

## 2. Présentation du projet

Site de réservation pour tatoueur avec idées de tatouage et présentation des projets déjà effectué.

---

## 3. Structure du site

---

### Page 1

Page principale du site :

- La grande bannière avec photo du salon, avec le logo et le nom
- Petite présentation du tatoueur
- Une partie réalisation avec des images des projets réalisés
- une seconde parti avec les flashs disponibles pour avoir des idées

---

### Page 2

Page secondaire du site :

- Une partie réalisation avec tous les plus beaux projets effectués
- Une partie secondaire avec des idées de tatouages :
  - Des boutons pour naviguer entre chaque thème
  - Plusieurs thèmes pour des idées avec des images

---

### Page 3

Dernière page du site :

- Page de réservation 
- Petite description des prérequis pour se faire tatouer (age, ...)
- Toutes les informations à remplir
- un bouton envoyé


**Challenge** : Faire un envoi de formulaire fictif, avec un message pour dire, c'est bien envoyé
